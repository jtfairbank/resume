How to Print / Export to PDF
============================

All Systems
-----------
Grab /out/resume-latest.pdf and print that.

Mac OSX
-------
 1. Open resume.md in TextEdit.
 2. Make the font 3 sizes smaller (everywhere).
 3. Print or export to PDF. A 'print preview' is recommended.
